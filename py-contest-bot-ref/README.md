Prerequisites
------------

  * TwitterAPI
  * Python 2.7
  
Configuration
------------

Open up `config.json` and make the values correspond to your Twitter API credentials.

Installation
------------
From the command line:

	pip install TwitterAPI
	
Then run:

	python main.py
